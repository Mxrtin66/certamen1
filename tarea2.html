Explora los conceptos básicos de HTML y Python con ejemplos visuales.

1. ¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es el estándar para crear páginas web. Permite estructurar el contenido y definir elementos como títulos, párrafos, imágenes y enlaces.

2. ¿Qué es Python?

Python es un lenguaje de programación potente y fácil de aprender, ampliamente usado en ciencia, web, automatización y más.
